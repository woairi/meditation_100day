# 🧘 명상 100일 챌린지

100일 동안 하루 10분, 명상 습관을 만드는 PWA입니다. 로그인 없이 기기(localStorage)에만 기록이 저장됩니다.

## 기능

- **명상 타이머** — 5/10/15/20분 선택, 시작/종료 종소리, 화면 잠금 상태에서도 정확한 타임스탬프 기반 타이머
- **호흡 가이드** — 기본(4-4) / 이완(4-6) / 박스 호흡(4-4-4-4) 리듬의 원형 애니메이션, 명상 중 컨트롤 자동 숨김
- **배경 사운드** — 빗소리, 파도, 시냇물, 숲/새소리, 모닥불, 싱잉볼, 백색/갈색소음 (Web Audio API 합성, 오디오 파일 없음, 스테레오 입체감)
- **100일 가이드** — 호흡 기초 → 바디스캔 → 자애 명상 → 감사 등 10단계 테마별 일일 안내 문구
- **인증 기록** — 완료 도장 100칸 그리드, 월별 달력, 스트릭/총 명상 시간/최근 7일 통계
- **인증 카드 공유** — "Day N 완료" 이미지를 생성해 SNS·메신저로 공유
- **명상 일지** — 매 세션 후 소감 기록 및 수정
- **백업/복원** — 기록을 JSON 파일로 내보내고 다른 기기에서 가져오기
- **PWA** — 홈 화면에 설치 가능, 오프라인 동작, 명상 중 화면 꺼짐 방지(Wake Lock)

## 실행

빌드 도구가 필요 없습니다. 정적 서버로 서빙하면 됩니다:

```bash
python3 -m http.server 8000
# http://localhost:8000 접속
```

> 서비스 워커와 ES 모듈 때문에 `file://`로는 열 수 없습니다.

`?dev` 쿼리를 붙이면 세션이 10초로 줄어 테스트하기 편합니다: `http://localhost:8000/?dev`

## 배포 (GitHub Pages)

모든 경로가 상대 경로라 별도 설정 없이 동작합니다:

1. 저장소 Settings → Pages → "Deploy from a branch" 선택
2. 브랜치와 루트(`/`) 지정 후 저장

파일을 변경해 배포할 때는 `sw.js`의 `CACHE_NAME` 버전을 올려야 클라이언트 캐시가 갱신됩니다.

## 동작 규칙

- **Day N은 진행도 기준**: 완료한 날 수 + 1. 하루 걸러도 가이드 내용이 밀리지 않습니다.
- **스트릭은 달력 기준**: 하루 빠지면 연속 기록이 끊깁니다.
- **하루 한 번만 도장**: 같은 날 두 번째 세션은 "자유 명상"으로 도장 없이 진행됩니다.
- **세션 복구**: 명상 중 앱이 꺼져도 1시간 이내 재진입 시 이어서 할 수 있습니다.

## 알려진 한계

- iOS에서 사용자가 직접 화면을 잠그면 배경 사운드가 멈춥니다 (타이머는 정확하게 유지됨). Wake Lock이 화면을 켜둔 상태에서는 발생하지 않습니다.
- 브라우저 사이트 데이터를 삭제하면 기록이 사라집니다.
