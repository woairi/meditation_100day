# 🧘 명상 100일 챌린지

100일 동안 하루 10분, 명상 습관을 만드는 PWA입니다. 로그인 없이 기기(localStorage)에만 기록이 저장됩니다.

## 기능

- **명상 타이머** — 3~60분 프리셋 및 직접 입력, 시작/종료 종소리, 중간 종(절반·5분·10분), 화면 잠금 상태에서도 정확한 타임스탬프 기반 타이머
- **호흡 가이드** — 기본(4-4) / 이완(4-6) / 박스 호흡(4-4-4-4) / 4-7-8 / 코히런트(5.5-5.5) 리듬의 원형 애니메이션, 명상 중 컨트롤 자동 숨김
- **배경 사운드** — 빗소리, 파도, 시냇물, 숲/새소리, 모닥불, 싱잉볼, 백색/갈색소음 (Web Audio API 합성, 오디오 파일 없음, 스테레오 입체감), 단계에 어울리는 사운드 추천
- **100일 가이드** — 호흡 기초 → 바디스캔 → 자애 명상 → 감사 등 10단계 테마별 일일 안내 문구, 100일 완주 후·자유 명상에는 날마다 순환하는 지속 명상 안내
- **가이드 낭독** — 브라우저 내장 음성(Web Speech API)으로 안내 문구를 읽어주기 (선택). 오디오 파일 없음
- **오늘의 한 줄** — 홈에 매일 바뀌는 짧은 명상 경구
- **일일 알림** — 정해진 시각에 명상을 잊지 않도록 리마인더 알림 (오늘 이미 완료했으면 울리지 않음)
- **마음 상태 기록** — 명상 전후 기분을 5단계로 남겨 일지에서 변화를 돌아보기 (모두 선택 사항)
- **인증 기록** — 완료 도장 100칸 그리드, 월별 달력, 스트릭/총 명상 시간/최근 7일 통계, 주간 명상 시간 추이 차트
- **공유 카드** — "Day N 완료" 인증 카드와 여정 진척·마일스톤(10·30·50·100일) 카드를 생성해 공유
- **명상 일지** — 매 세션 후 소감 기록·수정, 소감/제목 검색 및 단계별 필터
- **백업/복원** — 기록을 JSON 파일로 내보내고 다른 기기에서 가져오기, 백업이 오래되면 안내
- **첫 방문 안내** — 챌린지 규칙(진행도·스트릭·백업)을 소개하는 온보딩 (최초 1회)
- **달력 색 밴드** — 100칸 그리드를 10단계 테마 색으로 구분해 여정을 한눈에
- **테마** — 다크/라이트 자동 전환(`prefers-color-scheme`)
- **접근성** — 타이머 스크린리더 안내, 모달 Esc 닫기·포커스 트랩
- **PWA** — 홈 화면에 설치 가능, 오프라인 동작, 명상 중 화면 꺼짐 방지(Wake Lock)

## 실행

빌드 도구가 필요 없습니다. 정적 서버로 서빙하면 됩니다:

```bash
python3 -m http.server 8000
# http://localhost:8000 접속
```

> 서비스 워커와 ES 모듈 때문에 `file://`로는 열 수 없습니다.

`?dev` 쿼리를 붙이면 세션이 10초로 줄어 테스트하기 편합니다: `http://localhost:8000/?dev`

## 배포 (GitHub Pages)

모든 경로가 상대 경로라 별도 설정 없이 동작합니다. `main` 브랜치에 push하면 GitHub Actions(`.github/workflows/deploy-pages.yml`)가 자동으로 `gh-pages` 브랜치에 미러링합니다.

1. 저장소 Settings → Pages에서 Source를 "Deploy from a branch"로, 브랜치는 `gh-pages` / 루트(`/`)로 지정
2. 이후 `main`에 push하면 자동 배포되며, Actions 탭에서 `workflow_dispatch`로 수동 실행도 가능합니다

파일을 변경해 배포할 때는 `sw.js`의 `CACHE_NAME` 버전을 올려야 클라이언트 캐시가 갱신됩니다.

## 동작 규칙

- **Day N은 진행도 기준**: 완료한 날 수 + 1. 하루 걸러도 가이드 내용이 밀리지 않습니다.
- **스트릭은 달력 기준**: 하루 빠지면 연속 기록이 끊깁니다.
- **하루 한 번만 도장**: 같은 날 두 번째 세션은 "자유 명상"으로 도장 없이 진행됩니다.
- **세션 복구**: 명상 중 앱이 종료돼도(예: 전화 수신) 1시간 이내 재진입 시 중단 지점부터 이어서 할 수 있습니다.
- **화면을 꺼도 계속 진행**: 화면을 끄거나 앱을 잠시 전환해도 타이머는 멈추지 않고 계속 흐릅니다. 명상을 멈추려면 일시정지 버튼을 사용하세요.

## 알려진 한계

- 화면이 꺼져 있는 동안에는 OS가 자바스크립트와 오디오를 멈추므로, 종료 종소리는 화면을 다시 켜는 순간 울립니다 (남은 시간은 정확히 유지됨).
- iOS에서 사용자가 직접 화면을 잠그면 배경 사운드가 멈춥니다 (타이머는 정확하게 유지됨). Wake Lock이 화면을 켜둔 상태에서는 발생하지 않습니다.
- 브라우저 사이트 데이터를 삭제하면 기록이 사라집니다.
- **알림은 브라우저·설치 여부에 따라 동작이 다릅니다.** 앱을 닫아도 울리는 백그라운드 알림은 Periodic Background Sync를 지원하는 환경(주로 홈 화면에 설치한 안드로이드 Chrome)에서만 동작하며, 그 외에는 앱을 열어 둔 동안에만 알림이 표시됩니다. iOS는 홈 화면에 설치한 PWA에서만 알림을 지원합니다.
